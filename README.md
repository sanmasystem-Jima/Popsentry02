Popsentry02
Windows11のデスクトップに現れる不要なポップアップ通知を、あなたの手で選別し、自動で消し去るためのツールです。

特徴
手動選別システム: 新しいポップアップが出た際、「はい（良い）」か「いいえ（いらない）」を選択するダイアログが表示されます。

ブラックリスト学習: 一度「いらない」と判定したポップアップはブラックリストに登録され、次回からは表示された瞬間に自動で消去されます。

ホワイトリスト維持: 「良い」と判定した通知は、これまで通り表示されます。

常時監視: スタートアップに登録しておくことで、作業を邪魔する通知を逃さず監視します。

使い方
Popsentry02.exe を実行します。

Windows11のスタートアップに登録して使用することをお勧めします。

ポップアップが表示されたら、ダイアログで選別を行ってください。
